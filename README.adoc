=== Publiceren nieuwe versie

Zorg dat je ingelogd bent:

$ docker login

Met de repository root als current working directory, bouw de nieuwe image:

$ docker build -t ghcr.io/netbeheer-nederland/dataproducten:latest .

Push de nieuwe image naar GHCR:

$ docker push ghcr.io/netbeheer-nederland/dataproducten:latest
