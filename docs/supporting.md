# For Modelers

* **[URI Policy](https://modellen.netbeheernederland.nl/uri-policy)**
    <br>Guideline document in which a strategy is laid out for minting URIs in a standardized and uniform way.
* **[Modeling Guidelines](https://modellen.netbeheernederland.nl/modeling-guidelines)**
    <br>This document overviews the grid modelling approach and details the grid model data requirements for data supplied by Dutch transmission and distribution system operators.
* **[Solution Designs](https://modellen.netbeheernederland.nl/solution-designs)**
    <br>This is a collection of solution designs for various data products created and maintained by Netbeheer Nederland.
* **[Architecture Decision Records](https://modellen.netbeheernederland.nl/adrs)**
    <br>Document which captures important architectural decisions including context, considerations and consequences.

## Reference models

* **[NBNL Object Register](https://modellen.netbeheernederland.nl/object-register)**
  <br>Name authority list for the NBNL Profile Group.

* **[TC57CIM](https://modellen.netbeheernederland.nl/tc57cim)**
  <br>The Common Information Model (CIM) is an electric power transmission and distribution standard developed by the electric power industry.

* **[CGMES](https://modellen.netbeheernederland.nl/im-cgmes)**
  <br>Energy management system application program interface (EMS-API) - Part 600-1: Common Grid Model Exchange Standard (CGMES) - Structure and rules.
