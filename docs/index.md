# Data Products

## NBNL Profile Group
* **[NBNL Equipment](https://modellen.netbeheernederland.nl/nbnl-equipment)**
  <br>Defines the electrical equipment in the network model, including their technical characteristics and relationships.
* **[NBNL Asset](https://modellen.netbeheernederland.nl/nbnl-asset)**
  <br>Describes the physical assets owned or managed by the utility, including their identity, lifecycle, and management information.
* **[NBNL Operation](https://modellen.netbeheernederland.nl/nbnl-operation)**
  <br>Captures the real-time operational state and control information of the power system used for monitoring and running the network.
* **[NBNL Geographical Location](https://modellen.netbeheernederland.nl/nbnl-geographical-location)**
  <br>Provides the geographic position and spatial representation of network assets and equipment on maps or geographic systems.
* **[NBNL Topology](https://modellen.netbeheernederland.nl/nbnl-topology)**
  <br>Represents an abstraction of the current electrical connectivity of the network for a specific use case.

## Use Cases
* **[Capaciteitskaart](https://modellen.netbeheernederland.nl/capaciteitskaart)**
  <br>The capacity heatmaps information model was defined to cover a superset of data exchanged by the NL system operators.
* **[Meetdata](https://modellen.netbeheernederland.nl/meetdata)**
  <br>The _Meetdata_ data product describes the metered data for a single or multiple market evaluation points (Aansluiting).
* **[Netburen](https://modellen.netbeheernederland.nl/netburen)**
  <br>This data product for EnergyHubs describes which information and data to exchange between DSO and customer/advisor when establishing an EnergyHub.
